# Task Tracker

This is a simple task tracker application.

## Files

- `main.go`: The main application logic.
- `main_test.go`: Unit tests for the application.
- `go.mod`: Go module dependencies.
- `tasks.json`: Stores the task data.

## How to Run

To run this application, ensure you have Go installed.

1. Navigate to the project directory:

   ```bash
   \golang-project\tasktracker
   ```

2. Run the application:

   ```bash
   go run main.go
   ```

## How to Test

To run the tests for this application:

```bash
go test ./...
```
